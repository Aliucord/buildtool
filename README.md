# Aliucord buildtool

Tool to build Aliucord and plugins for Aliucord.

### Usage

Save config.example.json to config.json and fill it out first

Build Aliucord:

```
./buildtool
```

Build a plugin:

```
./buildtool -p PLUGIN_NAME
```

Build all plugins:

```
./buildtool -p "*"
```
